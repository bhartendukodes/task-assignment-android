# Task Assignment - Android App

An Android companion app for the Task Assignment system, allowing users to manage and track tasks on the go.

## Features

- User login and authentication
- View assigned tasks
- Update task status
- Create new tasks (Admin/Manager)
- Push notifications for task updates
- Clean Material Design interface

## Tech Stack

- **Language:** Kotlin
- **Architecture:** MVVM
- **Networking:** Retrofit
- **UI:** Material Design / XML Layouts
- **Auth:** JWT Token-based
- **DI:** Hilt

## Getting Started

1. Clone the repository
```bash
git clone https://github.com/bhartendukodes/task-assignment-android.git
```
2. Open in Android Studio
3. Update the base URL in the network config to point to your backend
4. Build and run
